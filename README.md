# pat1-subtask3

Decimal → Binary Algorithm 

•	Keep dividing by 2 and save the remainders
•	 Take your decimal number
•	Divide by 2 : write down the remainder: 0 or 1  
•	Use the whole number answer and divide by 2 again
•	Repeat until you get to 0
•	Read the remainders backwards = your binary number

     Binary → Decimal Algorithm
•	Each position is worth double the previous one: 1, 2, 4, 8, 16...
•	Start from the rightmost digit of the binary number
•	Position values are: 1, 2, 4, 8, 16, 32... going left
•	If the digit is 1, add that position’s value to your total
•	If the digit is 0, add nothing
•	Add them all up = decimal answer
